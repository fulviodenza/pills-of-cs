# commands

commands available for this bot, for now are:

- `/help`: to get help;
- `/start`: to start the bot;
- `/pill`: to get a pill from my notebook;
- `/get_tags`: to get available tags to subscribe to;
- `/choose_tag [tag]`: to get subscribed to a specific tag;
